# 手势爱心粒子 ✋❤️

一个基于手势识别的交互式 3D 粒子艺术应用，用手势控制粒子在爱心与文字之间流动变幻。

---

## 效果预览

- **静止状态**：2400+ 粒子构成旋转的 3D 爱心
- **张开手掌**：粒子散开，拼出 LOVE YOU / MISS U / FOREVER 等文字
- **握拳**：粒子收束，重新凝聚成爱心

---

## 功能特性

### 手势控制
| 手势 | 效果 |
|------|------|
| ✋ 张开手掌 | 粒子散开，显示当前文字形状 |
| ✊ 握拳 | 粒子收束回爱心 |

### 可切换形状
- `LOVE YOU` — 默认文字
- `MISS U`
- `FOREVER`
- `✦` — 星形
- `○` — 圆环
- `✺` — 爆炸/烟花
- `〰` — 波浪

### 交互控制
- **鼠标拖拽 / 触屏滑动**：旋转 3D 爱心
- **滚轮**：捏合/展开粒子
- **色相、饱和度、亮度滑块**：实时调整粒子颜色
- **粒子数滑块**：调整密度（500 ~ 4000）

### 视觉效果
- 5 层粒子层级（核心光晕 → 内层 → 中层 → 壳层 → 表面）
- 基于深度的渲染与透明度，营造真实 3D 感
- 呼吸/脉动动画
- 粒子辉光效果
- 平滑过渡动画

## 快速开始

### 环境要求
- 现代浏览器（Chrome / Firefox / Edge / Safari）
- 支持 WebRTC 的摄像头（使用手势功能时需要）
- **HTTPS 或 localhost**（浏览器摄像头权限要求）

### 运行方式

无需安装任何依赖，直接打开 HTML 文件即可：

```bash
# 方式一：使用 Python 启动本地服务器（推荐）
python3 -m http.server 8080
# 然后访问 http://localhost:8080/index_1.html

# 方式二：直接用浏览器打开文件
# 双击 index_1.html（摄像头功能可能受限）
```

### 使用步骤

1. 在浏览器中打开 `index_1.html`
2. 欣赏旋转的 3D 爱心粒子效果
3. 点击 **"✋ 开启摄像头 · 手势控制"** 按钮启用手势识别
4. 在摄像头前做手势：
   - 张开手掌 → 粒子散开成文字
   - 握拳 → 粒子收束成爱心
5. 左上角切换不同文字/形状
6. 底部滑块调整颜色和粒子密度

## 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 Canvas | 2D 粒子渲染 |
| Vanilla JavaScript | 核心逻辑 |
| CSS3 | UI 样式 |
| [MediaPipe Hands](https://mediapipe.dev/) | 实时手部姿态检测 |
| MediaPipe Camera Utils | 摄像头接入 |

### 核心算法
- **心形曲面**：参数方程生成 3D 心形坐标
- **3D 投影**：自定义旋转矩阵 + 透视投影
- **粒子物理**：弹簧运动 + 碰撞检测
- **手势判断**：基于 MediaPipe 21 个手部关键点计算张合度
