# 手势爱心粒子 ✋❤️

一个基于手势识别的交互式 3D 粒子艺术应用，用手势控制粒子在爱心与文字之间流动变幻。

---

## 效果预览

- **静止状态**：2400+ 粒子构成旋转的 3D 爱心
- **张开手掌**：粒子散开，拼出文字或几何形状
- **握拳**：粒子收束，重新凝聚成爱心

---

## 文件说明

| 文件 | 说明 |
|------|------|
| `index_1.html` | 基础版（桌面端，张开/握拳两种手势） |
| `index_countfingers.html` | 增强版（桌面端，多种手势自动切换形态） |
| `index_mobile.html` | 移动端适配版（响应式布局，触屏优化） |

## 功能特性

### 手势控制

#### index_countfingers.html（多手势版）

形态完全由手势自动控制，无需点击按钮：

| 手势 | 效果 |
|------|------|
| ☝️ 食指单伸 | 粒子拼出 MISS YOU |
| ✌️ 剪刀手（食指+中指） | 粒子拼出 LOVE YOU |
| 🤟 摇滚手（食指+小指） | 粒子拼出 FOREVER |
| ✋ 张开手掌（4指以上） | 粒子拼出 FOR YOU SIR ALWAYS |
| ✊ 握拳 | 粒子收束回爱心 |

#### index_1.html / index_mobile.html（基础手势版）

| 手势 | 效果 |
|------|------|
| ✋ 张开手掌 | 粒子散开，显示当前选中的文字/形状 |
| ✊ 握拳 | 粒子收束回爱心 |

### 可切换形状（基础版手动选择）

- `LOVE YOU` — 默认文字
- `MISS U`
- `FOREVER`
- `✦` — 星形
- `○` — 圆环
- `✺` — 爆炸/烟花
- `〰` — 波浪

### 交互控制
- **鼠标拖拽 / 触屏滑动**：旋转 3D 爱心
- **滚轮**：捏合/展开粒子
- **色相、饱和度、亮度滑块**：实时调整粒子颜色
- **粒子数滑块**：调整密度（500 ~ 4000）
- **🔄 重置识别按钮**：重启摄像头识别

### 视觉效果
- 5 层粒子层级（核心光晕 → 内层 → 中层 → 壳层 → 表面）
- 基于深度的渲染与透明度，营造真实 3D 感
- 呼吸/脉动动画
- 粒子辉光效果
- 平滑过渡动画

## 快速开始

### 环境要求
- 现代浏览器（Chrome / Firefox / Edge / Safari）
- 支持 WebRTC 的摄像头（使用手势功能时需要）
- **HTTPS 或 localhost**（浏览器摄像头权限要求）

### 运行方式

无需安装任何依赖，直接打开 HTML 文件即可：

```bash
# 使用 Python 启动本地服务器（推荐）
python3 -m http.server 8080
# 然后在浏览器访问对应页面
```

### 使用步骤

**多手势版（桌面推荐）：**
1. 访问 `index_countfingers.html`
2. 点击 **"✋ 开启摄像头 · 手势控制"** 启用识别
3. 用不同手势控制粒子形态（食指/剪刀/摇滚/张开/握拳）
4. 底部滑块调整颜色和粒子密度

**移动端：**
1. 访问 `index_mobile.html`
2. 点击启用摄像头
3. 张开手掌散开粒子，握拳收束成爱心
4. 左上角按钮手动切换形状

**基础版：**
1. 访问 `index_1.html`
2. 欣赏旋转的 3D 爱心
3. 开启摄像头后用张开/握拳控制粒子
4. 左上角切换文字/形状，底部调整颜色和粒子密度

## 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 Canvas | 2D 粒子渲染 |
| Vanilla JavaScript | 核心逻辑 |
| CSS3 | UI 样式 |
| [MediaPipe Hands](https://mediapipe.dev/) | 实时手部姿态检测 |
| MediaPipe Camera Utils | 摄像头接入 |

### 核心算法
- **心形曲面**：参数方程生成 3D 心形坐标
- **3D 投影**：自定义旋转矩阵 + 透视投影
- **粒子物理**：弹簧运动 + 碰撞检测
- **手势判断**：基于 MediaPipe 21 个手部关键点计算各手指弯曲度（`fingerCurl`），识别张合状态与具体手势
